# 📡 Network Traffic Capture and Analysis Using Wireshark

## 🎯 Objective
The objective of this project is to capture live network packets and analyze different network protocols using Wireshark. This helps in understanding how data travels through networks and how different protocols operate.

---

## 🛠 Tool Used
- **Wireshark** – Free and open-source network protocol analyzer.

---

## 🔍 Steps Performed

1. Installed Wireshark.
2. Started packet capture on the active network interface.
3. Generated network traffic by browsing websites and pinging a server.
4. Captured packets for about one minute.
5. Stopped the capture.
6. Applied filters to identify different protocols.
7. Exported the packet capture as a `.pcap` file.
